Flask Scanner Pro

User Guide (English)

1. Introduction

Welcome to Flask Scanner Pro. This system is designed for Rupasinghe Trust Investments Ltd to digitize pawn bills efficiently using AI-powered OCR (Optical Character Recognition). The system processes dual-document A4 scans, links related records, and stores data securely in the cloud.

2. Batch Scanning Process

The Batch Scan module allows you to upload scanned A4 pages containing two vertically stacked documents (Receipt and Detail Bill). The system automatically splits the image, identifies the document type, and extracts fields like IR Number, Receipt Number, Customer Name, and NIC.

3. Verification and Linking

Once scanned, records appear in the verification queue. The system uses the IR/R Number to match the Primary Receipt with the Detail Bill. You can view images side-by-side to verify accuracy and manually correct any OCR mistakes before final saving.

4. Searching and Viewing Records

The Records page provides a comprehensive list of all digitized bills. You can search by Branch, IR Number, Receipt Number, or Customer Name. Detailed views show all extracted data along with the original document images hosted on Cloudinary.

5. System Settings

In the Settings panel, administrators can manage Gemini API keys for OCR, update the list of branches, and configure storage parameters. Ensure at least one valid API key is active for the scanning process to work.
